# Changelog

All notable changes to the "Modern Dark Pro" extension will be documented in this file.

## [1.0.1] - 2026-04-14

### Changed
- Improved extension icon with a transparent background and enhanced design.

## [1.0.0] - 2026-04-10

### Added
- Initial release of Modern Dark Pro
- Modern Dark Pro - Monokai variant with vibrant accent colors
- Modern Dark Pro - Night variant optimized for extended work sessions
- Comprehensive syntax highlighting for:
  - JavaScript/TypeScript
  - HTML/CSS
  - Python
  - Java
  - C/C++
  - Go
  - Rust
  - And many more languages
- Optimized terminal colors for both variants
- Git integration colors with professional styling
- Professional editor UI with refined borders and accents
- Complete documentation and installation guide

### Design Features
- Carefully tuned dark background colors for minimal eye strain
- High-contrast syntax highlighting for excellent readability
- Professional accent colors (blue, green, orange, red, purple)
- Support for semantic token coloring
- Bracket pair colorization support
- Git diff colors optimized for clarity
