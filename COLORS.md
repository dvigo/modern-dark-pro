# 🎨 Modern Dark Pro - Color Palette

## Monokai Variant Colors

### Primary Backgrounds
- **Editor**: `#272822` - Main editor background
- **Sidebar**: `#272822` - Sidebar background
- **Panel**: `#272822` - Bottom panel
- **Status Bar**: `#3e3d32` - Status bar

### Accent Colors
- **Blue (Primary)**: `#66d9ef` - Links, focus, selections
- **Green**: `#a6e22e` - Success, additions, HTML tags
- **Yellow**: `#e6db74` - Strings, constants
- **Orange**: `#fd971f` - Classes, types
- **Red**: `#f92672` - Keywords, errors
- **Purple**: `#ae81ff` - Functions, special keywords
- **Cyan**: `#a1efe4` - Special tokens
- **Pink**: `#f8f8f0` - Text

### Text Colors
- **Foreground**: `#f8f8f2` - Main text
- **Comment**: `#75715e` - Comments (italic)

## Syntax Highlighting

### Both Themes
- **Keywords** (if, else, return): Red / Pink
- **Functions**: Purple
- **Classes/Types**: Orange
- **Strings**: Yellow
- **Constants**: Yellow / Blue
- **Comments**: Gray, italic
- **Variables**: Orange / Text color
- **HTML Tags**: Green
- **Attributes**: Blue

## Design Philosophy

1. **Professional appearance**: Clean, minimal aesthetic
2. **High contrast**: Excellent readability for extended sessions
3. **Consistent theming**: Same semantic colors across variants
4. **Eye comfort**: Optimized background colors to reduce strain
5. **Modern standards**: Support for semantic token coloring

---

**Note**: Modern Dark Pro focuses on professional development with careful color selection for both form and function.
